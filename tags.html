Some of the important tags are written below , Note that these tags are written inside "< >"

  • h1 to h6 - tags are used for heading

  • The text size in h1 is largest and keeps on decreasing upto h6

  • p -tag is used for writting a paragraph

  • img -tag is used for inserting an image

  • ul -tag is used for unordered list

  • div -tag is used for dividing the layout of whole page and specifying different class to different portions

  • span -tag is used for grouping of inline-elements

  • It does not make any visual change by itself

  • It is very similar to the div tag but div is a block-level tag and span is an inline tag.
