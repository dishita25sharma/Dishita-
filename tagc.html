Some important CSS tags are:

  • color : Used to change the font color

  • font-size : Used to change the font size

  • font-family : Used to change the font family

  • background-color : Used to change the background color

  • width : Used to change the width

  • height : Used to adjust the height

  • background-image : Used to change the background image

  • background-size : Used to adjust the cover

  • margin : Used to allign the text at the center

  • letter-spacing : Used to insert space in text

  • hover : Used to highlight the text

  • border-radius : Used to round the corners of element

  • text-align : Used to align the text to the left

  • text-right : Used to align the text to the right

  • opacity : Used to make the element transparet

  • rgba : Used to make the background ranceparent
